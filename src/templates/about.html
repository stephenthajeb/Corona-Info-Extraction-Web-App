Info Extraction App

Info Extractor is an app designed to extract information about date and numeric based on a keyword. It's purpose is to help people to quickly access latest update about Corona Virus. Input:multiple text files(English or Indonesian,CASE-INSENSITIVE). The algorithm used in processing the extraction :

  • Knuth Morris Path (KMP) Algorithm

    This algorithm used a lps (longest proper prefix which is also suffix) table to preprocess. The value of the obtained from the table can determine the position of test string to rollback while a mismatch happened between the test string and the pattern Worst case complexity for this algorithm is O(n)

  • Boyer Moore (BM) Algorithm

    This algorithm use a bad heuristic table for preprocessing. The bad heuristic contained key and value pair that determined the number of shift need to be done when a mismatch is happening. The table is obtained from the pattern and not the text. This algorithm match string from to left.

  • Regular Expression (Regex)

    Regular Expression is a sequence of symbol or character or metacharacter that defines a string or a grammar for a string. Regular Expression is mainly used to extract the numeric and date information in this app.
